# 비디오 디렉토리

이 디렉토리에는 웹사이트에서 사용할 비디오 파일들을 저장합니다.

## 권장 비디오 사양

- **포맷**: MP4 (H.264 코덱)
- **해상도**: 1920x1080 (Full HD) 또는 3840x2160 (4K)
- **프레임레이트**: 30fps 또는 60fps
- **비트레이트**: 5-10 Mbps (웹 최적화)
- **파일 크기**: 가능한 50MB 이하로 최적화

## 필요한 비디오 파일

1. **hero-video.mp4**: 메인 페이지 히어로 섹션 배경 비디오
   - 내용: 항공기, 화물선, 물류 창고 등 물류 관련 영상
   - 길이: 30-60초 (루프 재생)
